# 时间-FG%衰减曲线设计

## 1. 定位

一张小型折线图，放在 Hexbin 全场图的上方选择器栏中。展示投篮命中率随比赛进程的衰减趋势，支持点击/拖拽联动到 Hexbin 空间视图。

---

## 2. 视觉编码

| 维度 | 编码方式 | 说明 |
|---|---|---|
| X 轴 | 比赛时间（8 个 bin） | Q1前 → Q1后 → Q2前 → Q2后 → Q3前 → Q3后 → Q4前 → Q4后 |
| Y 轴 | 命中率 FG%（左轴）+ 出手次数（右轴，可选半透明柱状图） | 双轴避免柱状图遮挡折线趋势 |
| 线条 | 两条折线 | 蓝色=2PT FG%，橙色=3PT FG%，实线=左半场数据，虚线=右半场数据 |
| 选中高亮 | 点击的 bin 填充色加深 + 垂直虚线标记 | 明确指示当前 Hexbin 过滤到的时段 |
| 参考线 | 一条水平虚线 | 该实体的全时段平均 FG%，帮助判断"这个时段是高于还是低于平均" |

---

## 3. 数据聚合

### 时间分箱

从原始字段合成比赛绝对秒数，切分为 8 个 bin：

```python
seconds_elapsed = period * 12 * 60 - (minutes_remaining * 60 + seconds_remaining)
bin_index = seconds_elapsed // 360  # 0-7
```

| Bin | 标签 | 时间范围 |
|---|---|---|
| 0 | Q1 前 | 0–360s |
| 1 | Q1 后 | 360–720s |
| 2 | Q2 前 | 720–1080s |
| 3 | Q2 后 | 1080–1440s |
| 4 | Q3 前 | 1440–1800s |
| 5 | Q3 后 | 1800–2160s |
| 6 | Q4 前 | 2160–2520s |
| 7 | Q4 后 | 2520–2880s |

> 不含加时（Period ≥ 5），加时投篮量仅占 0.72%，独立处理后放入单独的数据点（可选）。

### 聚合逻辑

按时间 bin 分组，计算：
- `fg_pct`: SUM(Shot Made Flag) / COUNT(*)
- `fg2_pct`: 同上，仅 2PT Field Goal
- `fg3_pct`: 同上，仅 3PT Field Goal
- `total_shots`: COUNT(*)

所有聚合与当前选择器状态（球员/球队/联盟、赛季）一致。左右半场各跑一次聚合，得到两条独立的曲线数据。

## 4. 交互

### 点击选择

用户点击某个时间 bin → 该 bin 高亮（填充色加深 + 垂直虚线），Hexbin 视图过滤到该时间段的投篮数据。

**默认状态（无选中）：** 没有任何 bin 被选中，Hexbin 显示全时段数据。曲线仅作为概览存在——每条线上方显示一个小的总 FG% 数值标签。

### 拖拽区间选择

用户可以在 X 轴上拖拽一个区间（如从 Q1 前拖到 Q2 后）→ Hexbin 显示该区间内所有投篮的聚合。区间外的 bin 半透明显示。

### 右键 / 再次点击取消

再次点击已选中的 bin → 取消选中，恢复全时段视图。

### 悬停提示

悬停某个 bin 时，tooltip 显示：
```
Q4 后6min (最后6分钟)
全联盟 FG: 43.7%
2PT: 47.8% | 3PT: 33.4%
投篮次数: 572,338
```

### 联动模式

衰减曲线自身不设独立开关。其时间 bin 的点击/拖拽作用于**当前焦点所在的 Hexbin**（即用户最后操作过的那个半场的选择器）。焦点判断逻辑：

- 用户最近点击了左侧选择器（球员/球队下拉）→ 时间联动仅作用于左侧 Hexbin
- 用户最近点击了右侧选择器 → 时间联动仅作用于右侧 Hexbin
- 默认（页面加载时）焦点在左侧

被聚焦的半场 Hexbin 按选中时段过滤；非聚焦的半场保持全时段数据不变。若两个半场选择了同一实体（同球员/同球队），时间联动行为上与之前的"同步模式"等效——用户只需将时间 bin 分别应用于两侧即可。

> 去掉开关的原因：开关引入了"全局状态"概念，用户需要额外心智负担去理解当前处于哪种模式。改为跟随选择器焦点后，行为由"用户最后碰了哪个半场"自然决定，无需额外 UI。

## 5. 基于真实数据的参考值（全联盟，1997-2020）

| Bin | FG% | 2PT% | 3PT% | 出手数 |
|---|---|---|---|---|
| Q1 前 | 46.9% | 48.9% | 38.5% | 615,335 |
| Q1 后 | 46.3% | 48.9% | 37.0% | 608,162 |
| Q2 前 | 44.7% | 47.6% | 34.8% | 588,579 |
| Q2 后 | 46.3% | 49.3% | 36.3% | 590,400 |
| Q3 前 | 45.4% | 48.2% | 36.1% | 587,794 |
| Q3 后 | 44.8% | 47.8% | 35.9% | 569,456 |
| Q4 前 | 43.5% | 46.9% | 34.1% | 556,445 |
| Q4 后 | 43.7% | 47.8% | 33.4% | 572,338 |

> Q4 后比 Q4 前的 FG% 略反弹（43.5→43.7%），这是因为 Q4 后 6 分钟包含最后时刻的罚球和战术犯规后的篮下出手，拉高了总 FG%。但三分命中率仍然在跌（34.1→33.4%），说明疲劳对远投的影响是持续恶化的。

## 6. 实现细节

### ECharts 配置要点

```javascript
option = {
  xAxis: {
    type: 'category',
    data: ['Q1前', 'Q1后', 'Q2前', 'Q2后', 'Q3前', 'Q3后', 'Q4前', 'Q4后'],
    axisLabel: { rotate: 0 }
  },
  yAxis: [
    {
      type: 'value', name: 'FG%', min: 25, max: 55,
      axisLabel: { formatter: '{value}%' }
    },
    {
      type: 'value', name: '出手数', min: 0,
      axisLabel: { show: false }, splitLine: { show: false }
    }
  ],
  series: [
    // 2PT 折线（左半场实线）
    { type: 'line', data: left2PTData, color: '#5470C6', lineStyle: { width: 2 } },
    // 3PT 折线（左半场实线）
    { type: 'line', data: left3PTData, color: '#F59E0B', lineStyle: { width: 2 } },
    // 2PT 折线（右半场虚线）
    { type: 'line', data: right2PTData, color: '#5470C6', lineStyle: { type: 'dashed', width: 1.5 } },
    // 3PT 折线（右半场虚线）
    { type: 'line', data: right3PTData, color: '#F59E0B', lineStyle: { type: 'dashed', width: 1.5 } },
    // 柱状图（出手数）
    { type: 'bar', data: totalShots, yAxisIndex: 1, itemStyle: { opacity: 0.15 } }
  ]
}
```

### Pinia Store 联动

- `selectedTimeBin`: number | null（null=全时段）
- `activeHexbinFocus`: 'left' | 'right'（跟随用户最后操作的选择器，默认 'left'）

衰减曲线组件点击/拖拽 bin 时，检查 `activeHexbinFocus` 决定将 `selectedTimeBin` 作用于哪个半场的 Hexbin。另一侧 Hexbin 保持全时段数据。

## 7. 核验清单

- [ ] 无联动时（默认）折线显示全时段数据趋势，无选中高亮
- [ ] 点击 Q1 前 bin → 高亮 + Hexbin 显示 Q1 前时段投篮
- [ ] 再次点击已选中 bin → 取消高亮 + Hexbin 恢复全时段
- [ ] 左右半场在 3PT% 差距 > 3pp 时，折线可见明显分离
- [ ] 球员切换后，曲线在 300ms 内更新完毕
- [ ] 拖拽区间选择正确过滤 Hexbin 数据
- [ ] 时间联动跟随选择器焦点（左侧选择器操作→左侧 Hexbin 过滤；右侧→右侧过滤），默认焦点左侧
