# NBA 投篮可视分析系统 · 多视图交互与联动方案 v2

> 五个视图：Hexbin 热力图 · KDE 双峰相变 · 桑基投篮流向 · 三分转型分层 · 时间-FG%衰减曲线
> 设计原则：**按分析层次分页，页内近缘视图联动，跨页共享选中状态**

---

## 1. 五视图定位

| # | 视图 | 分析维度 | 核心问题 | 最佳搭档 |
|---|------|---------|---------|---------|
| A | **Hexbin 热力图** | 空间 | 球场上哪里最危险？谁在哪儿投？ | 衰减曲线 / 桑基图 |

## 2. 分页方案

### 2.1 三页结构

```
┌─────────────────────────────────────────────────────────────┐
│  顶部导航栏                                                  │
│  [🏀 空间探索]  [📈 演化趋势]  [🔀 投篮结构]                 │
│                                                             │
│  右侧常驻: 当前赛季标签 | 当前实体标签(可点×清除)             │
└─────────────────────────────────────────────────────────────┘

Page 1: 空间探索 (Space Explorer)
  ┌─────────────────────┐
  │  时间-FG% 衰减曲线   │  ← 上方控制条（8个时间bin可点击）
  ├─────────────────────┤
  │                     │
  │  Hexbin 全场热力图   │  ← 下方主图
  │                     │
  └─────────────────────┘
  联动: 点击衰减曲线bin → Hexbin过滤到该时段
        Hexbin选区域 → 衰减曲线标注该区域的平均命中率参考线

Page 2: 演化趋势 (Evolution Trends)
  ┌──────────┬──────────┐
  │  KDE     │  三分转型 │
  │  双峰相变 │  分层图   │
  │          │          │
  └──────────┴──────────┘
  联动: KDE播放赛季 → 三分转型竖线跟随
        三分转型点球队 → KDE叠加该队距离曲线（若有数据）

Page 3: 投篮结构 (Shot Structure)
  ┌──────────┬──────────┐
  │  桑基图   │  Hexbin  │
  │  投篮流向 │  热力图   │
  │          │          │
  └──────────┴──────────┘
  联动: 桑基图点区域节点 → Hexbin高亮对应区域六边形
        桑基图点Shot Type → Hexbin过滤2PT/3PT
        Hexbin点区域 → 桑基图高亮经过该区域的流向
```

### 2.2 为什么这样分组

| 分组 | 依据 | 不必放在一起的 |
|------|------|--------------|
| Hexbin + 衰减曲线 | 衰减曲线设计文档明确要求放在 Hexbin 上方。两者共享空间视角 | 桑基图和衰减曲线放一起→页内太多内容 |
| KDE + 三分转型 | 共享时间轴，是系统仅有的两个宏观时序视图 | Hexbin放这里→失去空间探索的独立性 |
| 桑基图 + Hexbin | 流程→空间的映射是五种视图中最直观的因果链 | 衰减曲线放这里→三视图页内太拥挤 |

### 2.3 导航设计

顶部固定导航栏，三个 Tab + 共享状态指示器：

```
┌──────────────────────────────────────────────────────────────┐
│ 🏀 NBA Shot Evolution                                       │
│                                                              │
│ [● 空间探索]  [  演化趋势  ]  [  投篮结构  ]       📅 2015-16 │
│                                                       👤 联盟 │
└──────────────────────────────────────────────────────────────┘
```

- 点击 Tab 切换页面，带淡入过渡动画（200ms）
- 右侧显示当前共享的赛季和实体，所有页面一致
- 切换页面时，共享状态保留（不会因为换页而重置选中）

## 3. 全局共享状态（Pinia Store）

这是三页之间的"记忆"。切换页面时，之前的选中不被丢弃。

```typescript
// stores/analysisContext.ts

interface AnalysisContext {
  // ═══ 跨页共享 ═══
  selectedSeason: string              // "2015-16"
  selectedTeamId: number | null       // 1610612744
  selectedTeamName: string | null     // "Golden State Warriors"
  selectedPlayerId: number | null
  selectedPlayerName: string | null

  // ═══ Page 1 专属: 空间探索 ═══
  selectedTimeBin: number | null      // 0-7, null=全时段
  timeLinkMode: 'sync' | 'independent' // 衰减曲线联动模式

  // ═══ Page 2 专属: 演化趋势 ═══
  isKdeAnimating: boolean
  kdeCurrentIndex: number             // 当前播放帧
  selectedAdoptionGroup: 'leader' | 'laggard' | 'mid' | null

  // ═══ Page 3 专属: 投篮结构 ═══
  selectedZone: string | null         // Shot Zone Basic
  selectedShotType: '2PT' | '3PT' | null

  // ═══ 联动追踪 ═══
  lastChangeSource: string            // 哪个视图触发了最近一次变更
}
```

### 状态生存周期

```
切换 Tab:
  - 共享状态（season, team, player）→ 保留
  - 页面专属状态（如 selectedTimeBin）→ 保留（用户切回来时还是之前选的那个bin）
  
点击导航栏 × 清除实体:
  - selectedTeamId/PlayerId → null
  - 所有页面恢复到"联盟平均"视角

浏览器刷新:
  - 全部重置为默认值（2019-20赛季，联盟平均）
```

### 4.2 联动逻辑

#### L1-1: 衰减曲线 → Hexbin（时间→空间）

```
用户在衰减曲线点击 Q4后 bin
  │
  ├─→ store.selectedTimeBin = 6
  │
  ├─→ 衰减曲线: Q4后 bin 高亮（填充色加深 + 垂直虚线）
  │
  └─→ Hexbin: 
       左右半场均过滤为 Q4后6分钟的投篮数据
       可观察到 Q4后6分钟：
         - 三分占比显著高于Q1（关键时刻更倾向三分，但命中率更低）
         - 中距离出手减少
         - 总体命中率下降
```

**叙事价值**: "注意看Q4最后6分钟，三分占比反常升高——但命中率反而是最低的。关键时刻球员选择了更低效的投篮。这是人性的弱点。"

#### L1-2: 衰减曲线拖拽区间 → Hexbin

```
用户在衰减曲线上从 Q3后 拖拽到 Q4后
  │
  └─→ Hexbin 显示整个下半场（Q3后半+Q4全部）的投篮聚合
```

#### L1-3: Hexbin 选区域 → 衰减曲线标注

```
用户在 Hexbin 点击 "Mid-Range" 区域气泡
  │
  ├─→ Hexbin: 中距离区域高亮
  │
  └─→ 衰减曲线:
       叠加一条新的水平参考虚线 = 该区域的全时段平均 FG%（39.4%）
       用户可以看到中距离在任何时段都低于平均命中率
       标注："中距离全时段均值: 39.4%"
```

#### L1-4: 衰减曲线同步/分开模式

```
同步模式 (默认):
  点击 bin → 左右两个 Hexbin 同时过滤到该时段
  用于: 对比两个球员在同一时段的投篮分布

分开模式:
  点击 bin → 仅左侧 Hexbin 过滤，右侧保持全时段
  用于: 左侧看 Q4 关键时刻，右侧看全时段基准 → 发现差异
```

#### L1-5: Hexbin 切换赛季 → 衰减曲线更新

```
用户在 Hexbin 选择 1997-98 赛季
  │
  ├─→ store.selectedSeason = "1997-98"
  │
  └─→ 衰减曲线:
       数据切换为 1997-98 赛季的聚合
       可对比 1998 vs 2020 的衰减模式是否不同
```

|------|---------|------|

| `timeLinkMode` | Pinia Store | Page1 专属 |
| `selectedSeason` | Pinia Store | 跨页共享 |
| `selectedTeamId` / `selectedPlayerId` | Pinia Store | 跨页共享 |

### 5.2 联动逻辑

#### L2-1: KDE 播放 → 三分转型竖线跟随（主联动）

```
KDE 播放动画，从 1998 逐帧推进到 2020
  │
  ├─→ store.selectedSeason 每 500ms 更新一次
  │
  ├─→ KDE: 白色曲线过渡到当前赛季
  │
  └─→ 三分转型:
       竖虚线跟随 KDE 当前赛季移动
       竖线经过 2012-2015 区间时 → 可以看到 leader 组的三分占比正在飙升
       竖线经过 2018 时 → 可以看到 laggard 组开始陡增
```

**这是整个系统最具叙事张力的联动**——KDE 展示"投篮分布怎么从单峰变成双峰"，三分转型图同时展示"与此同时，哪些球队在推动这个变化"。

#### L2-2: 三分转型点击球队 → KDE 叠加（深层联动）

```

       若有 GSW 的球队级 KDE 数据 → 叠加第四条虚线（Curry蓝）
       标注 "GSW 2019-20 距离分布: 篮下31% 中距12% 三分32%"
       若球队级 KDE 不可用 → 不做改变 + 轻提示"球队级距离分布暂不可用"
       
       叙事: "勇士的投篮分布比联盟平均更加极端——中距离只占12%"
```

  │

       在时序图上叠加一条水平虚线 = 中距离出手占比的联盟均值变化
       可以看到这条虚线在 1998-2020 间持续下降

### 6.2 联动逻辑

#### L3-1: 桑基图点击区域节点 → Hexbin 高亮（主联动）

```
用户在桑基图点击 "Above the Break 3"（弧顶三分）节点
  │
  ├─→ store.selectedZone = "Above the Break 3"
  │
  ├─→ 桑基图: "Above the Break 3" 节点 + 其下游流向高亮
  │
  └─→ Hexbin:
       球场热力图中，弧顶三分区域的六边形用闪烁边框高亮
       其他区域的六边形降低透明度
       右侧信息栏显示该区域的统计
```

#### L3-2: 桑基图点击 Shot Type → Hexbin 过滤

```
用户在桑基图点击 "3PT Field Goal" 节点
  │
  ├─→ store.selectedShotType = "3PT"
  │
  ├─→ 桑基图: 三分相关的流向高亮
  │
  └─→ Hexbin:
       热力图仅显示三分投篮
       两分区域（禁区、中距离）的六边形变为极淡
       标题更新: "三分投篮空间分布 — 2015-16"
       叙事: "三分出手集中在底角和弧顶，禁区三分几乎为零"
```

#### L3-3: 桑基图点击 Outcome → Hexbin 高亮

```
用户在桑基图点击 "Made"（命中）节点
  │
  └─→ Hexbin:
       热力图切换为"命中率"模式——每个六边形仅显示命中的投篮
       或者：显示命中-未中的差值热力图（命中率热力图）
       "看到哪些区域是真正的'甜点区'"
```

#### L3-4: Hexbin 点击区域气泡 → 桑基图高亮（反向联动）

```
用户在 Hexbin 点击 "Mid-Range" 气泡
  │
  ├─→ store.selectedZone = "Mid-Range"
  │
  ├─→ Hexbin: 中距离区域高亮
  │
  └─→ 桑基图:
       "Mid-Range" → "2PT Field Goal" → 各种Action的流向高亮
       可以看到中距离投篮绝大部分以 Missed 结束
       右侧信息栏: "中距离 | 命中率 39.4% | 占比从28%降到14%"
```
